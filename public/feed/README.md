# Feed images (generated)

PNG files here are produced by:

```bash
npm run generate:images
```

Requires `OPENAI_API_KEY` in the project root `.env`. Filenames must match `image` paths on items in `src/App.jsx` (`MIXED_FEED`).

Do not commit API keys. You may commit generated PNGs if you want the repo to show images without running the script.
